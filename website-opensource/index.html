Get Started in Minutes

Clone, configure, and deploy

1

Clone the Repository

git clone https://github.com/yourusername/ai-social-media-manager.git cd ai-social-media-manager
2

Install Dependencies

cd backend && npm install cd ../frontend && npm install
3

Configure Environment

cp .env.example .env # Edit .env with your configuration
4

Set Up Database

npm run prisma:generate npm run migrate npm run seed
5

Start Development

npm run dev # Backend: http://localhost:3001 # Frontend: http://localhost:3000
